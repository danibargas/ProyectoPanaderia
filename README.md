🥐 Proyecto Panadería

  Aplicación web para la gestión de una panadería, que permite a los usuarios registrarse, iniciar sesión y realizar compras de productos online.

🚀 Funcionalidades

  👤 Registro e inicio de sesión de usuarios
  🛒 Compra de productos
  📦 Gestión de pedidos
  ✏️ CRUD de productos (crear, editar, eliminar)
  🔐 Gestión básica de autenticación

🛠️ Tecnologías utilizadas

  Backend: C#
  Base de datos: SQL
  Frontend: HTML, CSS, Sass

⚙️ Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en local:

1. Clonar el repositorio:

```bash
git clone https://github.com/danibargas/ProyectoPanaderia.git
```

2. Acceder a la carpeta del proyecto:

```bash
cd ProyectoPanaderia
```

3. Configurar la base de datos:

  Crear una base de datos en SQL
  Importar el script incluido (si aplica)
  Configurar la cadena de conexión en el proyecto

4. Ejecutar el proyecto desde el IDE (Visual Studio recomendado)
